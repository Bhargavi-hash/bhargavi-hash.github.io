CoMeT: Towards Code Mixed Machine Translation Using Parallel Monolingual Sentences

Generating a machine translation system for English to Hinglish in supervised learning. In supervised learning, the algorithm “learns” from the training dataset by iteratively making predictions on the data and adjusting for the correct answer. (Here we are making use of supervised learning because we are using labelled data so that we can get the entire context at the encoder.) The translation systems could be used to augment datasets for various Hinglish tasks by translating datasets from English to Hinglish
